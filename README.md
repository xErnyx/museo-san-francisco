# 🏛️ San Francisco Museum - Web Development Project

This repository features a modern, interactive website dedicated to the **San Francisco Museum** in Quito, Ecuador. This project was developed to highlight local cultural heritage through an optimized digital experience.

## 🔗 Live Demo
**[Explore the Project Here](https://xernyx.github.io/museo-san-francisco/)**

</div>

### 🛠️ Technical Stack
The project utilizes professional frontend tools to ensure high performance and modularity:
*   **Core**: HTML5, CSS3, and JavaScript (ES6+).
*   **Bundler**: **Webpack** (Custom configurations for development and production).
*   **CI/CD**: **GitHub Actions** for automated deployment.
*   **Hosting**: GitHub Pages.

---

### 🚀 Key Features
*   **Responsive Design**: Fully optimized for seamless viewing across mobile, tablet, and desktop devices.
*   **Integrated Blog**: A functional section dedicated to museum news and historical research.
*   **Modular Architecture**: Clean code management using Webpack configurations.
